## The Review Flywheel

How Climbo compounds value over time. Each piece connects to the next.

```
Customer gets SMS/email
        ↓
Taps link → leaves Google review
        ↓
SEO Agent writes keyword-rich reply  →  ranks higher on Maps
        ↓
Social Agent posts it to GBP  →  more activity signal
        ↓
GEO Agent publishes it as content  →  appears in ChatGPT / AI Overviews
        ↓
More visibility → more walk-ins → more customers → more reviews
```
