## Overview

Onboarding is not optional. The Skool community says it plainly:

> "Onboarding is not optional — it's a must. The better you get your clients set up, the longer they stay and the better they pay." — Community post, 26 comments

A client you sign but don't properly onboard is a churn risk within 30 days. A client who sees results in week one stays for years.

## How it works

### Before the client's first call (Day 0)

- [ ] Create the client in your Climbo dashboard (`Clients → Add Client`)
- [ ] Assign the correct plan
- [ ] Apply any coupon code if they're on a trial
- [ ] Send the Welcome Email (customize the template first — see [[operations/onboarding-clients]])
- [ ] Book the onboarding call with a specific time. No vague "we'll connect soon." An actual calendar invite.

### Onboarding call (Day 1–2) — record it

This call does everything. It typically takes 45–60 minutes.

**Google Business Profile connection:**
- [ ] Client shares GBP access (you request Manager access via Google, or they do it with you on screen)
- [ ] Connect GBP to Climbo (`Settings → Integrations → Google`)
- [ ] Verify all GBP information is accurate: hours, phone, address, services, photos

**Review collection setup — pick the right method:**

| Business type | Best method |
|--------------|-------------|
| Restaurant, salon, gym | QR code kiosk + NFC card at reception |
| Professional services (lawyer, dentist) | Manual quick request via WhatsApp/SMS after each appointment |
| Business with a CRM | Auto-send via Zapier/Google Sheets integration |

- [ ] Generate and print the QR code page (Climbo generates it automatically — print and laminate)
- [ ] Order/configure the NFC card if using one (see [[features/nfc-cards]])
- [ ] Create at least one review request template (SMS, WhatsApp, or Email)
- [ ] Test the review link: open it on your phone, confirm it shows the client's branding

**AI Agents:**
- [ ] Enable SEO Agent — set language, keywords (business name + city + main service)
- [ ] Enable Social Agent — connect Facebook page and/or Instagram
- [ ] Enable GEO Agent if on a plan that includes it
- [ ] Run "reply to all past reviews" if the client has existing unanswered reviews

**Establish the baseline:**
- [ ] Screenshot current Google review count and rating ("before" number)
- [ ] Note the date. This is Day 1 of your results tracking.

### Day 7 — proactive touch

- [ ] Take a screenshot of any new reviews that have come in
- [ ] Send it to the client via WhatsApp without being asked
- This takes 2 minutes. It is worth its weight in trust.

### Day 14 — check-in call

- [ ] Show the before/after review count
- [ ] Adjust anything not working (review collection volume too low? Change the trigger timing)
- [ ] Ask: "Who else do you know in a similar situation?"

### Day 90 — harvest

- [ ] Review count should have grown visibly
- [ ] Ask for referrals and any testimonials about working with you

## Key details / Tips

- The review collection process is **90% of the results.** If your client is not collecting reviews, nothing else matters. Verify it is working live on the call, not later.
- Train the client's staff to ask for the scan/review in person. The QR code sitting on the counter does nothing if employees don't mention it.
- Never promise specific review counts or ratings. Promise a working system.
- If the client is not tech-savvy, use the Climbo AI Chatbot option — they manage everything through a chat interface instead of a dashboard.
